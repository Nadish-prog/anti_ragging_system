# 📌 Overview

The **Anti-Ragging Compliance System** is a full-stack complaint management platform that enables educational institutions to handle ragging incidents in a structured and transparent manner.

The platform provides:

- A secure complaint submission system
- Investigation workflow management
- Administrative oversight
- Case tracking until resolution

This system ensures **accountability, transparency, and faster incident resolution**.

---

# 🚨 Problem

Many institutions still rely on **manual reporting systems**, which leads to:

- Underreporting of ragging incidents
- Delayed investigations
- Poor complaint tracking
- Lack of accountability

A centralized digital platform is required to manage complaints efficiently.

---

# 💡 Solution

This system introduces a **structured complaint lifecycle** where:

1. Students submit complaints.
2. Administrators review and assign investigators.
3. Faculty conduct investigations.
4. The system tracks progress until the case is resolved.

---

# ✨ Features

### 📩 Complaint Reporting
Students can securely report ragging incidents with detailed information.

### 👨‍💼 Admin Management
Admins can:

- Assign investigators
- Set severity levels
- Track case status

### 🧑‍🏫 Investigation Workflow
Faculty members can:

- Receive assigned complaints
- Update investigation progress
- Submit investigation results

### 🔐 Role-Based Access Control

| Role | Access |
|-----|------|
| Student | Submit complaints |
| Admin | Assign investigators & manage complaints |
| Faculty | Conduct investigations |
